# Excel-AgentAi-on-FastApi-server-
Hi, this is a very simple representation of what i can make in short time using standard Ai Agent Tools like LangGraph, LangChain, Ollama etc. 
The project focus on safe, locally made work on excel files, so you can work on private data without worrying of your data leaking into servers by Ai models
Project is still onGoing, on days there will be added RAG for more possibilities then just using simple tools, Connection to Azure PostreSQL for chat memory and realtime visual view of your exce; file also i will grow the graph more.
Project is small but its only for portfolio purposes, you can awate for some bigger projects i will public here :D
At README you will later find some informations about how to setup something like this on your computer
